# Gents Glamor

Gents Glamor is an e-commerce app where user can buy product, see there ordered product. Admin can add product and manage existing product.

# Technology I have used for client side

- React
- Firebase Authentication
- Firebase Hosting
- Bootstrap
- React Bootstrap
- Moment JS
- React Router
- React Hook Form
